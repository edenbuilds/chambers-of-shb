## Use Cases

### 1. Commercial Dispute Classification

**Determination Framework:**
```
IS THIS A COMMERCIAL DISPUTE?

1. SPECIFIED VALUE THRESHOLD
   Contract/Transaction Value: ₹___
   Specified Value ≥₹3 lakhs?: [Yes / No]

   If NO → Regular civil court (not Commercial Court)
   If YES → Proceed to Step 2

2. "COMMERCIAL DISPUTE" DEFINITION (Section 2(1)(c))
   Does dispute arise from:
   □ Transactions of merchants/bankers/financiers/traders
   □ Commercial contracts (distribution, licensing, franchising, joint venture, shareholders, tech development, IP with Specified Value)
   □ Infrastructure/construction contracts
   □ Carriage of goods
   □ Export/import
   □ Financing/banking/securities
   □ Insurance
   □ Partnership (excluding family arrangements)
   □ IT/software/e-commerce
   □ IP rights (with Specified Value)
   □ Listed company matters
   □ Admiralty/maritime

   EXCLUSIONS:
   □ Immovable property for exclusive residence use
   □ Consumer disputes (Consumer Protection Act)
   □ Oppression & mismanagement (Companies Act)
   □ Testamentary/succession matters

3. JURISDICTION
   If Specified Value ≥₹3 lakhs AND commercial dispute:
   - District-level: Commercial Court
   - High Court original jurisdiction areas: Commercial Division

RESULT: [Commercial Court / Regular Court]
```

### 2. Pre-Institution Mediation Compliance (Section 12-A)

**Mandatory Mediation Assessment:**
```
PRE-INSTITUTION MEDIATION REQUIREMENT

RULE (Section 12-A):
No commercial dispute suit (≥₹3 lakhs) shall be instituted unless plaintiff exhausts pre-institution mediation

EXCEPTIONS (Mediation NOT Required):
□ Urgent interim relief (Order XXXIX CPC)
□ Execution proceedings
□ Summary suits (Order XXXVII - negotiable instruments, debts)
□ Admiralty matters (vessel arrest)
□ Public interest affecting large number of people

PROCEDURE:
1. File application with Mediation Centre/Authority
2. Authority issues notice to respondent
3. Mediator appointed within 7 days
4. Mediation timeline: 3 months (extendable by 2 months = 5 months max)
5. If settled: Settlement agreement has status of arbitral award (executable)
6. If not settled: Certificate of non-settlement issued
7. Plaintiff can file suit attaching certificate

TIMELINE IMPACT:
- Mediation: 3-5 months
- Delays suit filing but overall faster if settled (avoids 18-24 month litigation)
- Success rate: ~30% (2022-2024 data)

STRATEGIC DECISION:
Mediation Viability: [High / Medium / Low]
Recommendation: [Attempt good faith mediation / Seek exception / File urgent interim relief application]

PROTOCOL REFERENCE:
- IL-COMMERCIAL-001: Section 12-A detailed procedures
```

### 3. Commercial Courts Strict Timelines Compliance

**Critical Deadlines:**
```
COMMERCIAL COURTS TIMELINE TRACKER

⚠️ ABSOLUTE DEADLINES (Extensions Limited):

1. WRITTEN STATEMENT: 120 DAYS MAXIMUM
   - Initial: 30 days from summons
   - Extension: Maximum 90 days (total 120 days)
   - Post-120 days: Right to file FORFEITED (SCG Contracts 2019)
   - NO condonation beyond 120 days

2. INTERIM APPLICATIONS: 30 DAYS DISPOSAL
   - Section 13(1): Court shall dispose within 30 days
   - Extension: Maximum 30 days (total 60 days)
   - Applies to: Order XXXIX (injunction), Order XXXVIII (attachment)

3. REPLICATION/REJOINDER:
   - Replication: 30 days (extendable to 60 days)
   - Rejoinder: 30 days (extendable to 60 days)

4. TRIAL TARGET: 12-18 MONTHS
   - Not statutory but administrative target
   - Actual average: 18-24 months (2022-2024 data)

5. APPEALS: 75% DEPOSIT REQUIREMENT
   - Section 13(1-A): Must deposit 75% of decree amount to file appeal
   - Or furnish security for 75%
   - Huge barrier to frivolous appeals

COMPLIANCE STRATEGY:
□ Diarize all deadlines immediately
□ Request extensions well before expiry (with reasons)
□ For written statement: File within 30 days if possible (don't rely on extensions)
□ For interim relief: Assume 30-day window (not 60 days)

PROTOCOL REFERENCE:
- IL-COMMERCIAL-001: Strict Timelines section
```

### 4. Case Management Hearings Strategy

**Case Management Protocol:**
```
CASE MANAGEMENT HEARING PREPARATION (Order XV-A CPC)

FIRST CASE MANAGEMENT HEARING (FCMH):
Timeline: Within 4 weeks of summons service
Purpose:
- Frame preliminary issues
- Identify undisputed facts
- Narrow disputed issues
- Fix timelines for pleadings, discovery, evidence

WHAT TO BRING:
□ Brief of case (1-2 pages: Facts, issues, relief)
□ Document list (what you have, what you need from opponent)
□ Witness list (tentative)
□ Timeline estimate (how long for discovery, evidence, trial)
□ Settlement prospects (court encourages mediation even at this stage)

COURT'S ROLE:
- Proactive case management (not passive umpire)
- Identify delaying tactics
- Fix realistic but firm deadlines
- Encourage settlement

SECOND CASE MANAGEMENT HEARING:
Timeline: After written statement filed
Purpose:
- Frame issues for trial (Order XIV)
- Review document disclosure compliance
- Fix evidence dates
- Limit witnesses if excessive
- Set trial completion date

CASE MANAGEMENT PLAN:
Court prepares written plan:
- Issues framed
- Documents to be disclosed
- Witnesses identified (with testimony brief)
- Evidence timeline (affidavits, cross-examination dates)
- Trial completion estimate

STRATEGIC TIPS:
1. Come prepared (shows seriousness, builds credibility)
2. Be realistic on timelines (don't promise what you can't deliver)
3. Narrow issues genuinely (don't create false disputes)
4. Consider settlement (Commercial Courts strongly encourage)

PROTOCOL REFERENCE:
- IL-COMMERCIAL-001: Case Management section
```

### 5. Summary Judgment Application (Order XIII-A)

**Summary Judgment Strategy:**
```
SUMMARY JUDGMENT ASSESSMENT (Order XIII-A CPC)

CONCEPT:
If defendant has "no real prospect of success" → Grant judgment for plaintiff without full trial

WHEN TO APPLY (Plaintiff):
□ Claim is straightforward (debt, negotiable instrument, clear contract breach)
□ Defendant's defense frivolous or hopeless
□ No genuine triable issues

STANDARD: "No real prospect of success"
- Not whether defendant will succeed (that's for trial)
- But whether defense is plausible or merely fanciful

PROCEDURE:
1. Plaintiff files application after defendant's written statement
2. Affidavit supporting application
3. Notice to defendant (30 days to respond)
4. Hearing
5. If granted: Full decree as if after trial
6. If refused: Proceeds to trial

SUCCESS RATE (2022-2024):
- ~30% applications granted
- Highest success: Debt recovery, negotiable instruments (50%+)
- Lowest success: Complex contractual interpretation

DEFENDANT'S DEFENSE:
To defeat summary judgment, show:
- Triable issues exist
- Defense has real (not fanciful) prospect
- Compelling reason for trial (credibility assessment, complex facts)

STRATEGIC CONSIDERATIONS (Plaintiff):
Benefits if granted:
- Save time (avoid 12-18 month trial)
- Save costs
- Immediate decree

Risks if refused:
- Wasted costs of application
- May antagonize court/defendant
- Trial proceeds anyway

Recommendation:
Apply for summary judgment: [Yes / No]
Basis: [Analysis of defense's real prospects]

PROTOCOL REFERENCE:
- IL-COMMERCIAL-001: Summary Judgment section
```
