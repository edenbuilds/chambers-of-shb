# Protocol: Patent Prosecution Highway (PPH)

**Protocol ID**: PROT-IPL-054
**Version**: 1.0.0
**Domain**: Indian Patent Law
**Category**: International Filing
**Last Updated**: 2026-01-16

---

## Purpose

Comprehensive guide to using Patent Prosecution Highway programs to expedite patent examination in India based on positive examination results from participating offices, and using Indian grants abroad.

---

## Activation Triggers

- Foreign patent granted seeking faster Indian prosecution
- Indian patent granted seeking foreign filing
- Parallel prosecution in multiple countries
- Expedited examination needed
- Portfolio harmonization strategy

---

## Prerequisites

- Understanding of PPH program structure
- Knowledge of participating offices
- Granted/allowed claims in first office
- Pending application in second office

---

## Core Process

### Step 1: Understanding PPH

#### What is PPH:

```
PATENT PROSECUTION HIGHWAY

Concept:
├── First office examines and allows claims
├── Applicant requests PPH in second office
├── Second office expedites examination
└── Uses first office work product

Benefits:
├── Faster examination (months vs. years)
├── Higher grant rate (historically)
├── Reduced examination burden
└── Cost savings (fewer OAs)
```

#### PPH Types:

| Type | Description |
|------|-------------|
| **IP5 PPH** | Among IP5 offices (JPO, EPO, USPTO, KIPO, CNIPA) |
| **Global PPH** | Broader participating offices |
| **PPH-MOTTAINAI** | PCT-based |
| **Bilateral PPH** | Country-to-country agreements |

### Step 2: India's PPH Status

#### Current Position (2024-2026):

| Aspect | Status |
|--------|--------|
| IP5 member | No |
| Global PPH participant | Limited |
| Bilateral agreements | Japan (JPO) pilot |
| PCT-PPH | No formal program |

#### India-Japan PPH Pilot:

```
INDIA-JAPAN PPH (PILOT)

Eligibility:
├── JPO grant/allowance
├── Corresponding Indian application
├── Claims substantially correspond
└── Application pending examination

Documents Required:
├── Request for PPH
├── JPO office actions (translated)
├── JPO allowed claims
├── Claim correspondence table
└── Request for Examination (if not filed)
```

### Step 3: PPH Request Requirements

#### General Requirements:

| Requirement | Description |
|-------------|-------------|
| **Office of Earlier Examination (OEE)** | Where claims allowed/granted |
| **Office of Later Examination (OLE)** | Where PPH requested |
| **Claim correspondence** | OLE claims ≤ OEE allowed claims |
| **Pending status** | Not yet examined in OLE |

#### Documentation:

```
PPH REQUEST DOCUMENTATION

1. PPH Request Form:
   - Application number (OLE)
   - Reference application (OEE)
   - Priority information

2. OEE Materials:
   - Office actions
   - Search reports
   - Allowance/grant decision
   - Allowed claims

3. Claims Correspondence:
   - Table showing claim mapping
   - OLE claim → OEE claim
   - Explanation of correspondence

4. Translations:
   - OEE documents in OLE language
   - Certified if required

5. Claim Amendment:
   - If narrowing claims to match OEE
```

### Step 4: Using Foreign Grants in India

#### Expedited Examination Alternative:

Since formal PPH is limited, consider:

```
LEVERAGING FOREIGN GRANTS IN INDIA

Option 1: Expedited Examination (Rule 24C)
├── If eligible category (startup, MSME, etc.)
├── Use foreign grant as persuasive evidence
└── Not formal PPH but helps

Option 2: Arguments in Prosecution
├── Cite foreign grants in response to OAs
├── Persuasive, not binding
├── Show consistency of claims
└── Address examiner's concerns

Option 3: PPH (Where Available)
├── Japan-India pilot
├── Check for new bilateral agreements
└── Follow specific procedures
```

### Step 5: Using Indian Grants Abroad

#### Countries Accepting Indian OEE:

| Country | PPH Available? |
|---------|---------------|
| Japan | Yes (pilot) |
| USPTO | Not direct (check Global PPH) |
| EPO | No formal PPH |
| CNIPA | No |
| KIPO | No |

#### Strategy:

```
INDIAN GRANT LEVERAGE ABROAD

1. Japan (via pilot):
   ├── File PPH request
   ├── Provide Indian grant documents
   └── Request expedited examination

2. Other Countries:
   ├── No formal PPH
   ├── BUT cite Indian grant
   ├── Persuasive in prosecution
   └── May influence examiner

3. PCT Strategy:
   ├── Positive ISR/IPRP
   ├── Based on Indian search authority
   └── Use in national phases
```

### Step 6: PPH Application Process

#### Step-by-Step:

```
PPH REQUEST PROCEDURE

1. Verify Eligibility:
   □ OEE application granted/allowed
   □ OLE application pending
   □ Claims correspond
   □ PPH program exists

2. Prepare Documents:
   □ PPH request form
   □ OEE office actions
   □ OEE allowed claims
   □ Claim correspondence table
   □ Translations

3. Amend Claims (if needed):
   □ Narrow to match OEE claims
   □ Remove broader claims
   □ Ensure correspondence

4. File Request:
   □ Submit to OLE
   □ Pay fees (if any)
   □ File RFE if not filed

5. Track Status:
   □ Acceptance of PPH request
   □ Examination timeline
   □ Office actions (if any)
```

### Step 7: Claim Correspondence Analysis

#### Requirement:

OLE claims must correspond to OEE allowed claims.

#### Correspondence Types:

| Type | Description |
|------|-------------|
| **Identical** | Same claims word-for-word |
| **Narrower** | OLE claims subset of OEE |
| **Substantively same** | Different wording, same scope |

#### Claim Mapping Table:

```
CLAIM CORRESPONDENCE TABLE

OLE (India)  |  OEE (Japan)  |  Correspondence
-------------|---------------|----------------
Claim 1      |  Claim 1      |  Identical
Claim 2      |  Claim 3      |  Narrower
Claim 3      |  Claims 1+2   |  Combination
Claim 4      |  Claim 4      |  Identical

Note: All OLE claims must find correspondence
in OEE allowed claims.
```

### Step 8: Strategic Considerations

#### When to Use PPH:

| Scenario | PPH Recommended? |
|----------|-----------------|
| Fast market entry needed | Yes |
| Claims already allowed elsewhere | Yes |
| Prosecution budget limited | Yes |
| Unique validity issues in India | Maybe not |
| Broader claims desired in India | No |

#### PPH vs. Other Options:

```
COMPARISON: PPH VS. EXPEDITED EXAMINATION

PPH:
├── Requires foreign grant
├── Claims must correspond (narrow)
├── Very fast examination
├── Higher grant likelihood
└── Limited availability in India

Expedited Examination (India):
├── No foreign grant needed
├── Full claim scope maintained
├── Requires eligibility (startup, etc.)
├── Moderately faster
└── Broadly available

Regular Examination:
├── No constraints
├── Full flexibility
├── Longer timeline
└── Standard process
```

## PPH Decision Framework

```
PPH SUITABILITY ANALYSIS

Have foreign grant?
├── No → Consider expedited examination
└── Yes ↓

PPH available between offices?
├── No → Cite foreign grant in prosecution
└── Yes ↓

Willing to narrow claims?
├── No → Standard prosecution
└── Yes ↓

Claims correspond?
├── No → Amend, then PPH
└── Yes → FILE PPH REQUEST
```

## Common PPH Mistakes

1. **Broader OLE claims**: Must be ≤ OEE claims
2. **No program available**: Check before relying
3. **Missing correspondence**: Incomplete mapping
4. **Wrong documents**: Need OEE specifics
5. **Timing issues**: OLE already examined
6. **Translation problems**: Certification requirements
