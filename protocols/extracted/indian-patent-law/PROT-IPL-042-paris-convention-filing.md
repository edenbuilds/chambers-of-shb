# Protocol: Paris Convention Filing Strategy

**Protocol ID**: PROT-IPL-042
**Version**: 1.0.0
**Domain**: Indian Patent Law
**Category**: International Filing
**Last Updated**: 2026-01-16

## Purpose

Comprehensive guide to using Paris Convention priority for filing patent applications in India and from India to foreign countries, including priority claim procedures, deadline management, and strategic considerations.

---

## Activation Triggers

- Filing convention application in India
- Claiming foreign priority for Indian filing
- Filing abroad from Indian priority
- 12-month priority deadline approaching
- Multiple country filing strategy

## Prerequisites

- Understanding of Paris Convention principles
- Priority application details
- 12-month deadline awareness
- Foreign filing license (if applicable)

## Core Process

### Step 1: Paris Convention Basics

#### Fundamental Principles:

| Principle | Meaning |
|-----------|---------|
| **Priority right** | 12-month window from first filing |
| **National treatment** | Same treatment as nationals |
| **Independence** | Each country's patent independent |

#### Priority Period:

```
First Filing (any Paris Convention country)
        = Priority Date
        ↓
12 months
        ↓
Deadline for Convention Filing
```

### Step 2: India as Convention Application

#### Section 135 - Convention Applications:

"[A]n application for a patent made by virtue of this section shall be made in the prescribed manner and shall be accompanied by a complete specification"

#### Requirements for Convention Filing in India:

| Requirement | Details |
|-------------|---------|
| Priority claim | Cite earlier application |
| 12-month window | From earliest priority date |
| Complete specification | Must accompany application |
| Priority document | Certified copy |
| Declaration | Of right to claim priority |

### Step 3: Filing Procedure

#### Convention Application to India:

**Required Documents**:
```
1. Form 1 (Application for grant of patent)
   - Mark as "Convention Application"
   - State priority application details

2. Complete Specification
   - Full disclosure
   - Claims
   - Abstract

3. Priority Document
   - Certified copy from priority office
   - OR DAS code access

4. Form 3 (Statement/undertaking)
   - Other applications filed

5. Power of Attorney (if applicable)
```

### Step 4: Priority Claim Requirements

#### Form 1 Declarations:

```
PRIORITY CLAIM IN FORM 1

Application Type: Convention

Priority Details:
├── Country: [Priority country]
├── Application No.: [Priority application number]
├── Filing Date: [DD/MM/YYYY]
└── Title: [Must match]

Multiple Priorities:
├── Priority 1: [Details]
├── Priority 2: [Details]
└── Priority 3: [Details]
```

#### Multiple Priorities:

- Can claim multiple priorities
- Each priority for different parts of specification
- Earliest priority determines deadline

### Step 5: Priority Document

#### Options for Providing:

| Method | Details |
|--------|---------|
| **Certified copy** | Paper or electronic from priority office |
| **DAS (Digital Access Service)** | WIPO system, access code |
| **Bilateral arrangement** | Between India and priority country |

#### DAS Procedure:

```
1. Request DAS code from priority office
2. Priority office uploads to WIPO DAS
3. Provide DAS code to Indian Patent Office
4. IPO retrieves document electronically
```

#### Timeline:
- Priority document: Within 3 months of request by Controller
- May request extension if justified

### Step 6: Filing from India to Foreign Countries

#### Pre-Filing Requirements:

**Section 39 - Foreign Filing License**:

Must either:
1. File in India first, wait 6 weeks, then file abroad
2. Obtain written permit from Controller before foreign filing

**Consequence of Violation**:
- Section 39 contravention
- Patent void in India
- Potential criminal liability

#### Claiming Indian Priority Abroad:

```
Indian Application (Day 0 - Priority Date)
        ↓
Wait 6 weeks (or get permit)
        ↓
File in foreign countries
        - Claim Indian priority
        - Within 12 months
        ↓
Provide certified copy of Indian application
```

### Step 7: Multiple Priority Claims

#### Scenario: Complex Priority Chain

```
Application 1 (Country A) - covers feature X
        ↓ (within 12 months)
Application 2 (Country B) - covers feature Y (new)
        ↓ (within 12 months from App 1)
Convention Filing India - covers X + Y
        - Claims priority from App 1 for X
        - Claims priority from App 2 for Y
```

#### Rules:
- Each claim gets priority from earliest application disclosing it
- 12-month period runs from earliest priority
- Must disclose priority claims

### Step 8: Strategic Considerations

#### Paris Route vs. PCT:

| Factor | Paris Convention | PCT |
|--------|------------------|-----|
| Deadline | 12 months | 30/31 months |
| Cost timing | Upfront per country | Deferred |
| Countries | Any Paris member | PCT members |
| Search report | None centralized | ISR |
| Best for | 1-3 countries, certainty | Multiple countries, uncertainty |

#### When to Use Paris Convention:

✅ **Prefer Paris When**:
- Filing in 1-3 countries only
- High certainty about markets
- Budget available upfront
- Non-PCT countries targeted

❌ **Prefer PCT When**:
- Many countries (4+)
- Need time to assess
- Budget-constrained
- Want ISR opinion

## Priority Deadline Calculator

### Formula:

```
Priority Date: [Date of first application]
        +
12 months
        =
Deadline for Convention Filing

Example:
Priority Date: January 15, 2025
Deadline: January 15, 2026
```

### Expiry Rules:

- If deadline falls on holiday/weekend: Next working day
- Month calculation: Same date next year
- If no same date: Last day of month

## Checklist: Filing Abroad with Indian Priority

### Pre-Filing:
- [ ] Section 39 compliance (6-week wait or permit)
- [ ] Foreign countries identified
- [ ] Within 12 months from Indian filing
- [ ] Agent in foreign country engaged

### Filing:
- [ ] Claim Indian priority
- [ ] Provide application number
- [ ] Provide filing date
- [ ] Request certified copy from IPO

### Post-Filing:
- [ ] Submit certified copy to foreign office
- [ ] File Form 3 in India (update)
- [ ] Track foreign prosecution

---

## Common Paris Convention Mistakes

1. **Missing 12-month deadline**: No extension available
2. **No foreign filing license**: Section 39 violation
3. **Wrong priority date**: Verify earliest application
4. **Missing priority document**: Request early
5. **Title mismatch**: Should correspond
6. **Not updating Form 3**: Compliance issue
