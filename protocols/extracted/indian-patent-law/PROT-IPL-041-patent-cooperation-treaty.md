## Core Process

### Step 1: PCT System Overview

#### What is PCT:

The Patent Cooperation Treaty provides a unified procedure for filing patent applications to protect inventions in member countries (157 contracting states as of 2025).

#### PCT Does NOT:

- Grant international patents
- Replace national filings (defers them)
- Guarantee patentability
- Bind national offices on grant

#### PCT DOES:

- Single international application
- International search report (ISR)
- Written opinion on patentability
- Defer national phase costs (30/31 months)
- Simplify multi-country filing

### Step 2: India in the PCT System

#### India's Roles:

| Role | Authority |
|------|-----------|
| **Receiving Office (RO)** | Accepts PCT applications |
| **International Searching Authority (ISA)** | Conducts prior art search |
| **International Preliminary Examining Authority (IPEA)** | Provides examination report |
| **Designated/Elected Office** | National phase entry |

#### India as ISA/IPEA:

**Accepts Applications From**:
- Indian residents/nationals
- Singapore residents (ISA/IPEA)
- Persian-speaking applicants (ISA)

**Benefits of Indian ISA**:
- Lower cost than EPO/USPTO
- English language
- Familiarity with Indian practice

### Step 3: Filing PCT from India

#### Pre-Filing Requirements:

**Foreign Filing License (Section 39)**:
- Required before PCT filing if invented in India
- Wait 6 weeks after Indian filing, OR
- Obtain written permit from Controller

#### Filing Route Options:

**Route 1: RO/IN (India as Receiving Office)**
```
File PCT at Indian Patent Office (RO/IN)
        ↓
Choose ISA (India/EPO/others)
        ↓
International search
        ↓
Publication at 18 months
        ↓
Optional: Chapter II (IPEA)
        ↓
National phase entry by 30/31 months
```

**Route 2: RO/IB (Direct to WIPO)**
```
File PCT directly at International Bureau
        ↓
(Same subsequent process)
```

### Step 4: ISA Selection Strategy

#### Available ISAs for RO/IN Applications:

| ISA | Advantages | Disadvantages |
|-----|------------|---------------|
| **India (ISA/IN)** | Low cost, English | Limited capacity |
| **EPO** | High quality, Europe-focused | Higher cost |
| **USPTO** | US-focused search | Higher cost |
| **JPO** | Japanese prior art | Language considerations |
| **KIPO** | Korean prior art | Language considerations |

#### Selection Factors:

| Factor | Consideration |
|--------|---------------|
| Target markets | Match ISA to key countries |
| Technology area | ISA expertise matters |
| Budget | Costs vary significantly |
| Speed | Processing times differ |
| Quality | Report influence on national phase |

### Step 5: PCT Timeline

#### Key Deadlines:

| Event | Deadline |
|-------|----------|
| **Priority date** | Day 0 |
| **PCT filing** | 12 months from priority |
| **Record copy to IB** | RO transmits in 1 month |
| **Search copy to ISA** | Within 1 month of filing |
| **ISR establishment** | 16 months from priority (or 9 from filing) |
| **International publication** | 18 months from priority |
| **Demand for IPEA** | 22 months from priority (or 3 from ISR) |
| **IPER establishment** | 28 months from priority |
| **National phase entry** | 30/31 months from priority |

### Step 6: PCT Forms

#### Key Forms for Indian Applicants:

| Form | Purpose |
|------|---------|
| **PCT/RO/101 (Request)** | Main PCT application form |
| **PCT/RO/134** | Fee calculation sheet |
| **PCT/IPEA/401 (Demand)** | Request for preliminary examination |
| **PCT/IB/304** | Designation changes |

## PCT Strategy Considerations

### For Indian Applicants:

#### When to Use PCT:

✅ **Use PCT When**:
- Filing in 3+ countries
- Need more time to assess commercial potential
- Budget planning required
- Market strategy uncertain

❌ **Direct Filing Better When**:
- Only 1-2 countries
- Urgent protection needed
- Budget-constrained
- Already have market certainty

### Claiming Priority:

```
Indian Application (Day 0)
        ↓
Within 12 months
        ↓
PCT Application (claims Indian priority)
        ↓
30/31 months from Indian filing
        ↓
National Phase Entry (multiple countries)
```

### Bypass Strategy:

For India specifically:
- Can bypass PCT and file directly
- May be faster for India-only protection
- PCT only needed if going international

- [ ] Priority application filed (optional)

- [ ] Target countries identified
- [ ] Budget allocated

### Filing:
- [ ] PCT/RO/101 completed
- [ ] Specification in proper format
- [ ] Claims, abstract, drawings
- [ ] Fees calculated and paid
- [ ] Power of attorney (if agent filing)

### Post-Filing:
- [ ] Record copy received by IB
- [ ] Search copy to ISA
- [ ] ISR received and analyzed
- [ ] Written opinion considered
- [ ] Chapter II decision made
- [ ] National phase deadlines tracked

## Common PCT Mistakes (Indian Applicants)

1. **No foreign filing license**: Section 39 violation
2. **Wrong priority claim**: Must be within 12 months
3. **ISA mismatch**: Not available for chosen RO
4. **Late national phase**: Missing 31-month deadline
5. **Translation delays**: Plan ahead
6. **Fee miscalculation**: Verify current fees
