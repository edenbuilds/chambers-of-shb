# Protocol: IP Due Diligence for Transactions

**Protocol ID**: PROT-IPL-050
**Version**: 1.0.0
**Domain**: Indian Patent Law
**Category**: Transactions
**Last Updated**: 2026-01-16

---

## Purpose

Comprehensive guide to conducting patent and IP due diligence for mergers, acquisitions, investments, licensing deals, and other transactions involving Indian patent rights.

---

## Activation Triggers

- M&A transaction involving patents
- Investment in patent-owning company
- Technology acquisition deal
- IP portfolio purchase
- Joint venture with IP contribution
- Pre-IPO IP assessment

---

## Prerequisites

- Understanding of transaction objectives
- Access to target company records
- Knowledge of patent search methods
- Familiarity with valuation approaches

---

## Core Process

### Step 1: Due Diligence Framework

#### Categories of Review:

| Category | Focus |
|----------|-------|
| **Ownership** | Title, chain of assignments |
| **Validity** | Maintenance, potential challenges |
| **Scope** | Claims, coverage, limitations |
| **Freedom-to-Operate** | Infringement risks |
| **Encumbrances** | Licenses, pledges, liens |
| **Litigation** | Past, pending, threatened |

### Step 2: Ownership Verification

#### Title Chain Analysis:

```
OWNERSHIP VERIFICATION

1. Patent Office Records:
   ├── Named owner matches seller
   ├── Assignment chain complete
   └── No gaps in title

2. Inventor Rights:
   ├── Employment agreements
   ├── Assignment documents
   └── No unresolved claims

3. Corporate Changes:
   ├── Mergers properly documented
   ├── Spin-offs addressed
   └── Name changes recorded

4. Geographic Coverage:
   ├── India patent ownership
   ├── Foreign counterparts
   └── PCT filings
```

#### Documentation Checklist:

| Document | Purpose |
|----------|---------|
| Original grant | Proof of patent |
| Assignment deeds | Title transfer |
| Employment agreements | Inventor assignment |
| Board resolutions | Authority to assign |
| Merger documents | Corporate succession |

### Step 3: Validity Assessment

#### Validity Risks:

| Risk Factor | Assessment |
|-------------|-----------|
| Prior art | Search for invalidating references |
| Section 3 issues | Patentability concerns |
| Prosecution history | File wrapper review |
| Opposition history | Past challenges |
| Maintenance fees | Current status |

#### Due Diligence Searches:

```
VALIDITY SEARCH SCOPE

1. Prior Art:
   - Pre-filing publications
   - Earlier patents
   - Public use evidence

2. Patent Status:
   - Renewal fees paid
   - No pending challenges
   - No compulsory license

3. Prosecution Quality:
   - Examiner objections addressed
   - No hidden prior art
   - Complete disclosure

4. Opposition History:
   - Pre-grant opposition outcome
   - Post-grant proceedings
   - Court challenges
```

### Step 4: Scope Analysis

#### Claims Review:

| Element | Analysis |
|---------|----------|
| Independent claims | Core protection scope |
| Dependent claims | Backup positions |
| Claim types | Product, process, system |
| Limitations | Narrow vs. broad |
| Prosecution amendments | Impact on scope |

#### Commercial Relevance:

```
CLAIM-PRODUCT MAPPING

Patent Claims:
├── Claim 1: [Core technology]
│   └── Products covered: [List]
├── Claim 5: [Variant]
│   └── Products covered: [List]
└── Claims 10-15: [Applications]
    └── Products covered: [List]

Revenue Attribution:
├── Direct coverage: [% of revenue]
├── Indirect protection: [% of revenue]
└── Future products: [Potential]
```

### Step 5: Encumbrances and Obligations

#### Types of Encumbrances:

| Type | Impact |
|------|--------|
| **Exclusive license** | Cannot grant another |
| **Non-exclusive license** | Multiple licensees |
| **Security interest** | Bank/lender rights |
| **Cross-license** | Reciprocal rights |
| **Settlement agreement** | Covenant not to sue |
| **Joint ownership** | Co-owner rights |

#### Encumbrance Review:

```
ENCUMBRANCE ANALYSIS

1. Existing Licenses:
   ├── Exclusive vs. non-exclusive
   ├── Field of use restrictions
   ├── Territory limitations
   ├── Royalty obligations
   └── Change of control clauses

2. Security Interests:
   ├── Registered pledges
   ├── Bank liens
   └── Release requirements

3. Third-Party Rights:
   ├── Government use rights
   ├── Research exemptions
   └── Compulsory license risk

4. Contractual Restrictions:
   ├── Non-compete clauses
   ├── Non-assertion covenants
   └── Most favored licensee
```

### Step 6: Litigation Review

#### Litigation History:

| Type | Review |
|------|--------|
| Past infringement | Outcome, claims tested |
| Pending litigation | Status, exposure |
| Opposition proceedings | Validity tested |
| Licensing disputes | Interpretation issues |
| Threatened claims | C&D letters |

#### Red Flags:

```
LITIGATION RED FLAGS

⚠️ Warning Signs:
├── Multiple validity challenges
├── Narrow claim interpretation by court
├── Settlement with questionable terms
├── Pending IPR/opposition
├── Threatened suits by NPEs
└── Expired litigation holds

Critical Issues:
├── Unresolved inventorship disputes
├── Pending compulsory license
├── Government use notification
└── Criminal IP proceedings
```

### Step 7: Freedom-to-Operate Review

#### For Acquirer's Products:

| Analysis | Purpose |
|----------|---------|
| Third-party patents | What others own |
| Infringement risk | Does target infringe |
| Design-around | Options available |
| Indemnification | Allocation of risk |

#### For Target's Products:

```
FTO ANALYSIS

Target Company Products:
├── Product A:
│   ├── Target's patents: [Coverage]
│   ├── Third-party risks: [List]
│   └── FTO opinion: [Status]
├── Product B:
│   ├── Target's patents: [Coverage]
│   ├── Third-party risks: [List]
│   └── FTO opinion: [Status]
└── Pipeline:
    ├── Development stage: [Products]
    ├── Known risks: [Analysis]
    └── Clearance status: [Pending/Complete]
```

### Step 8: Due Diligence Report

#### Report Structure:

```
IP DUE DILIGENCE REPORT

EXECUTIVE SUMMARY
- Key findings
- Risk assessment
- Recommendations

1. PORTFOLIO OVERVIEW
   - Patents and applications
   - Geographic coverage
   - Claim mapping

2. OWNERSHIP ANALYSIS
   - Title verification
   - Assignment chain
   - Outstanding issues

3. VALIDITY ASSESSMENT
   - Prosecution quality
   - Prior art risks
   - Maintenance status

4. SCOPE ANALYSIS
   - Claim breadth
   - Commercial relevance
   - Competitive position

5. ENCUMBRANCES
   - Existing licenses
   - Security interests
   - Restrictions

6. LITIGATION
   - Past and pending
   - Risk assessment
   - Exposure quantification

7. VALUATION INPUT
   - Strength assessment
   - Remaining life
   - Market relevance

8. RECOMMENDATIONS
   - Risk mitigation
   - Transaction adjustments
   - Post-closing actions
```

## Due Diligence Request List

### Documents to Request:

```
IP DUE DILIGENCE REQUEST LIST

A. PATENT DOCUMENTATION
   1. All Indian patents and applications
   2. Foreign counterparts
   3. PCT filings and national phases
   4. File histories/prosecution files
   5. Search reports and office actions

B. OWNERSHIP RECORDS
   6. Assignment agreements
   7. Employment/invention agreements
   8. Chain of title documents
   9. Inventor declarations
   10. Board resolutions for IP

C. LICENSES AND AGREEMENTS
   11. All license agreements (in and out)
   12. Cross-license agreements
   13. Settlement agreements
   14. Joint development agreements
   15. Research collaboration agreements

D. LITIGATION FILES
   16. Pending litigation documents
   17. Past litigation outcomes
   18. Opposition proceedings
   19. Cease and desist correspondence

E. FINANCIAL DATA
   21. Royalty income/expense

   23. IP insurance policies
   24. Valuation reports
```

## Risk Matrix

### Due Diligence Risk Assessment:

| Risk Category | Low | Medium | High |
|---------------|-----|--------|------|
| **Ownership** | Clear chain | Minor gaps | Disputed |
| **Validity** | Strong | Some risk | Challenged |
| **Scope** | Broad | Moderate | Narrow |
| **Encumbrances** | None | Non-exclusive | Exclusive |
| **Litigation** | None | Settled | Pending |
| **FTO** | Clear | Minor issues | Blocked |

## Common Due Diligence Mistakes

1. **Incomplete search**: Missing applications/foreign
2. **Ignoring prosecution**: File wrapper matters
3. **No FTO review**: Third-party risks overlooked
4. **Title assumptions**: No verification
5. **License review gap**: Hidden encumbrances
6. **Valuation overreach**: Strength not assessed
7. **Time pressure shortcuts**: Incomplete review
