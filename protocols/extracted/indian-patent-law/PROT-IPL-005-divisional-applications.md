# Protocol: Divisional Patent Applications

**Protocol ID**: PROT-IPL-005
**Version**: 1.0.0
**Domain**: Indian Patent Law
**Category**: Application Strategy
**Last Updated**: 2026-01-16

---

## Purpose

Comprehensive guide for filing and prosecuting divisional patent applications under Section 16 of the Patents Act, 1970, including the significant changes from the 2024 Amendment Rules and the Syngenta decision.

---

## Activation Triggers

- Unity of invention objection in examination report
- Desire to protect multiple inventions disclosed in parent
- Strategic portfolio expansion
- Continuation-style prosecution strategy

---

## Prerequisites

- Pending parent application (not yet granted)
- Multiple inventions disclosed in provisional or complete specification
- Understanding of claim scope strategy

---

## Core Process

### Step 1: Understand Legal Basis

#### Section 16 - Divisional Applications:
"A person who has made an application for a patent...may, at any time before the grant of the patent, if he so desires, or with a view to remedy the objection raised by the Controller on the ground that the claims of the complete specification relate to more than one invention, file a further application..."

#### Key 2024 Amendment (Rule 13(2A)):
Applicants may file one or more further applications under Section 16, including for an invention disclosed in:
- The **provisional specification**
- The **complete specification**
- A **further divisional application**

This aligns with the Delhi High Court **Syngenta decision (October 2023)**.

### Step 2: Determine Filing Basis

#### Traditional Basis (Pre-2024):
- Unity objection from Controller
- Claims relate to more than one invention

#### Voluntary Filing (Post-2024):
- **Suo moto** filing without unity objection
- Based on any invention disclosed in:
  - Provisional specification
  - Complete specification description
  - Previously filed divisional

### Step 3: Timing Requirements

| Condition | Deadline |
|-----------|----------|
| Before parent grant | Mandatory - cannot file after parent grants |
| Voluntary filing | Any time before parent grant |
| Controller-directed | Per direction in examination report |

**Critical**: Once parent patent is granted, divisional filing window closes.

### Step 4: Prepare Divisional Application

#### Documents Required:
1. **Form 1** - Mark as divisional application
   - Reference parent application number
   - Reference filing date of parent
2. **Form 2** - Complete specification
   - New claims directed to divisional invention
   - Description can be same as parent or modified
3. **Form 3** - Statement under Section 8
4. **Form 5** - Declaration of inventorship

#### Claim Requirements:
- Claims must relate to invention disclosed in parent
- Must be distinct from claims in parent
- Cannot broaden beyond parent disclosure
- No new matter permitted

### Step 5: Filing and Priority

#### Effective Filing Date:
- Divisional deemed filed on **parent's filing date**
- Preserves all priority benefits of parent

#### Priority Claim:
- Inherits priority from parent automatically
- Same convention priority (if any)

### Step 6: Prosecution of Divisional

1. **Independent prosecution**: Separate from parent
2. **Own RFE deadline**: 31 months from original priority
3. **Own examination**: May receive different prior art
4. **Own grant timeline**: Independent of parent

### Step 7: Strategic Uses

#### 1. Portfolio Expansion
- Extract multiple patents from single disclosure
- Cover different aspects of same innovation

#### 2. Continuation-Style Strategy
- Similar to US continuation practice (per 2024 Rules)
- File divisionals to continue prosecution
- Pursue alternative claim scope

#### 3. Design-Around Protection
- Cover competitor variations
- Block design-arounds of parent claims

#### 4. Technology Licensing
- Separate patents for different licensees
- Flexibility in portfolio management

## Comparison with US Continuation Practice

| Feature | India Divisional (Post-2024) | US Continuation |
|---------|------------------------------|-----------------|
| Basis | Distinct invention disclosed | Same invention |
| New Matter | Not permitted | Not permitted |
| Timing | Before parent grant | Before parent issues |
| Voluntary Filing | Yes | Yes |
| Claim Overlap | Must be distinct | Can overlap |

**Note**: India still doesn't have true continuation applications. Divisionals must cover **distinct** inventions from parent claims.

## Key Case Law

### Syngenta Ltd. v. Controller of Patents (Delhi HC, October 2023)
- Divisional can be based on disclosure in description, not just claims
- Overruled restrictive Boehringer interpretation
- Now codified in Rule 13(2A)

### Boehringer Ingelheim (Earlier Restrictive View - Overruled)
- Had limited divisionals to plurality in claims only
- No longer good law

## Quality Checklist

- [ ] Parent application still pending (not granted)
- [ ] Divisional invention disclosed in parent specification
- [ ] Claims distinct from parent claims
- [ ] No new matter added
- [ ] Form 1 properly references parent
- [ ] Filing date correctly recorded as parent's date
- [ ] RFE deadline calculated from original priority
- [ ] Form 3 compliance maintained

---

## Common Divisional Mistakes

1. **Filing after parent grant**: Application invalid
2. **Overlapping claims with parent**: Rejection
3. **Adding new matter**: Ground for rejection/revocation
4. **Missing RFE deadline**: Calculated from original priority
5. **Not referencing parent**: Procedural defect
