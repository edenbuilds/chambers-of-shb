# Indian IP Law: Trademarks Protection

## Purpose
Comprehensive guidance on Trade Marks Act 1999 and Trade Marks Rules 2017, covering registrability, application procedure, opposition, well-known marks, infringement, passing off, dilution, and trademark portfolio strategy in India.

## Activation Triggers
Use when: Trademark selection and clearance | Registration application | Opposition proceedings | Infringement dispute | Passing off action | Well-known mark status | Trademark licensing/assignment | Domain name disputes | Geographical indication conflict | Comparative advertising | Parallel import issues | Trademark renewal/restoration.

## Core Principles

### Section 2(1)(zb): "Trade Mark" Defined

**"A mark capable of being represented graphically and which is capable of distinguishing the goods or services of one person from those of others"**

**Essential Elements:**

1. **MARK** - Any sign (word, logo, shape, color, sound, smell - if graphically representable)
2. **GRAPHICALLY REPRESENTABLE** - Can be depicted on paper
3. **DISTINCTIVE** - Distinguishes source of goods/services

**Types of Marks:**

- **Word Marks** - TATA, RELIANCE, GOOGLE
- **Device Marks** - Logos (Apple logo, Nike swoosh)
- **Combined Marks** - Word + logo
- **Shape Marks** - Product shape (Coca-Cola bottle)
- **Color Marks** - Single color OR combination (Cadbury purple)
- **Sound Marks** - Yahoo yodel, Nokia tune
- **Smell Marks** - Fragrance (rare, difficult to represent graphically)

### Section 9: Absolute Grounds for Refusal

**Marks That CANNOT Be Registered:**

**(a) Not Capable of Distinguishing** - Generic, descriptive terms

Example: "SOAP" for soap ✗ | "DOVE" for soap ✓ (arbitrary/fanciful)

**(b) Devoid of Distinctive Character** - Common words/symbols

Example: "BEST QUALITY" for products ✗

**(c) Descriptive Marks** - Describe characteristics

Example: "FAST" for courier service ✗ | "FEDEX" ✓

**(d) Customary in Trade** - Common in industry

Example: "THE" for newspapers ✗

**(e) Simple Geometrical Shapes** - Triangle, circle ✗

**(f) Offensive/Prohibited** - Scandalous, immoral marks

Example: National emblems, religious symbols (without permission)

**Exception - Acquired Distinctiveness (Proviso):**

Even if descriptive/non-distinctive initially, if mark has acquired distinctiveness through use, can be registered

Example:
- "YELLOW PAGES" originally descriptive
- Through extensive use → Acquired distinctiveness
- Registrable ✓

**Evidence of Acquired Distinctiveness:**
- Market share, sales figures
- Advertising expenditure
- Consumer surveys showing recognition
- Duration of use

### Section 11: Relative Grounds for Refusal (Conflict with Prior Marks)

**Cannot Register if:**

**(a) Identical with Earlier Mark for Identical Goods/Services**

Example:
- Prior registration: APPLE for computers
- Application: APPLE for computers → Rejected

**(b) Similar to Earlier Mark for Similar/Identical Goods/Services, Likelihood of Confusion**

Test: Would consumer confuse origin?

**Similarity Assessment:**
- Visual (appearance)
- Phonetic (sound)
- Conceptual (meaning)

Example:
- Prior: TIDE for detergent
- Application: TYDE for detergent → Likely confusing → Rejected

**(c) Identical/Similar to Well-Known Mark (Even for Different Goods/Services)**

Example:
- GOOGLE is well-known for search engines
- Application: GOOGLE for bicycles → Rejected (dilutes well-known mark)

## Trademark Registration Procedure

### Phase 1: Trademark Search and Clearance

**Step 1: Comprehensive Search**

**Databases to Search:**
- Indian TM Registry (official database)
- State emblems, prohibited marks
- Company names (MCA database)
- Domain names
- Common law prior use (unregistered marks)

**Search Strategy:**
- Identical marks in same class
- Phonetically similar marks
- Visually similar marks
- Translation equivalents

**Step 2: Clearance Opinion**

**Risk Assessment:**
- **Low Risk:** No similar marks → Proceed
- **Medium Risk:** Similar marks in different classes → Assess conflict
- **High Risk:** Similar marks in same class → Reconsider OR prepare for opposition

### Phase 2: Filing Application

**Application Form: TM-A**

**Information Required:**

1. **Applicant Details** - Name, address, nationality
2. **Mark** - Representation (JPG for device marks)
3. **Goods/Services** - Description + Nice Classification class
4. **Claimed User Date** - Date of first use (if applicable)
5. **Priority Claim** - If claiming foreign priority (6 months under Paris Convention)

**Nice Classification:**
- 45 classes total
- Classes 1-34: Goods
- Classes 35-45: Services

Example:
- Class 25: Clothing, footwear
- Class 35: Advertising, business management

**Multiple Classes:**
- Can file single application for multiple classes
- Fee per class

**Fee:**
- Physical application: Rs. 4,500 per class (individual/startup) | Rs. 9,000 (others)
- E-filing: Rs. 4,500 per class (individual/startup) | Rs. 9,000 (others)

**Filing Receipt:**
- Application number issued immediately
- Can use "TM" symbol (not ® - reserved for registered marks)

### Phase 3: Examination

**Vienna Codification (For Device Marks):**
- Examiner searches device elements using Vienna codes
- Identifies conflicting logos

**Examination Report:**
- Issued within 12-18 months
- Objections raised (Section 9, 11 grounds)
- Prior conflicting marks cited

**Response to Examination Report:**
- Deadline: 1 MONTH (extendable)
- Options:
  - Amend specification
  - Disclaim non-distinctive elements
  - Argue distinctiveness
  - Evidence of acquired distinctiveness/use
  - Narrow goods/services

**Hearing:**
- If objections not resolved → Hearing before Registrar
- Oral arguments
- Final decision: Grant OR Refusal

### Phase 4: Publication and Opposition

**Acceptance:**
- Mark accepted for registration
- Published in Trade Marks Journal

**Opposition Period: 4 MONTHS (from publication)**

**Who Can Oppose:** Any person (Section 21)

**Grounds:**
- Section 9, 11 grounds (same as examination)
- Prior use/reputation
- Bad faith

**Opposition Procedure:**

1. **Notice of Opposition (TM-O)** - Filed within 4 months
2. **Counter-Statement** - Applicant responds (2 months)
3. **Evidence** - Opponent's evidence (2 months) → Applicant's evidence (2 months) → Opponent's reply (1 month)
4. **Hearing** - Oral arguments
5. **Decision** - Registrar decides (grant OR refuse)

**Timeline:** 2-4 years if opposed

### Phase 5: Registration

**If No Opposition OR Opposition Dismissed:**
- Certificate of Registration issued
- Effective date: Application filing date
- Valid for: 10 YEARS (renewable indefinitely)

**Rights:**
- Exclusive use of mark
- Right to sue for infringement
- Use ® symbol

## Trademark Rights and Infringement

### Section 28: Rights of Registered Proprietor

**Exclusive Rights:**
- Use mark on goods/services
- Prevent unauthorized use by others
- Assign, license mark

**Infringement (Section 29):**

**(1) Identical Mark, Identical Goods/Services**
- Use of identical mark on identical goods → Automatic infringement (no confusion proof needed)

Example: Using "ADIDAS" on shoes (when ADIDAS registered for shoes) → Infringement

**(2) Similar Mark, Similar Goods/Services + Likelihood of Confusion**
- Must prove likelihood of confusion or association

Test:
- Would average consumer confuse source?
- Factors: Visual/phonetic/conceptual similarity, nature of goods, channels of trade

Example:
- ZARA registered for clothing
- ZARO used for clothing → Likely confusion → Infringement

**(3) Well-Known Mark + Unfair Advantage/Detriment**
- Using mark identical/similar to well-known mark
- Takes unfair advantage of reputation OR
- Causes detriment to distinctive character/repute

Example:
- MICROSOFT well-known for software
- MICROSOFT used for bicycles → Dilution of repute → Infringement

**(4) Identical/Similar Mark + Advertisement**
- Advertising goods/services under mark → Infringement (even if no actual sale)

**(8) Applying Mark to Packaging/Material**
- Affixing mark to packaging → Contributory infringement

**Defenses (Section 30):**

**(a) Bona Fide Use of Own Name/Address** - Can use own name/address if not trademark use

Example: "Johnson & Johnson" can be used by person named Johnson for goods

**(b) Descriptive Use** - Bona fide use of descriptive terms

**(c) Comparative Advertising** - Honest comparison identifying competitor's mark (conditions apply)

**(d) Prior Continuous Use** - If used continuously before opponent's registration/application

Example:
- A used mark since 2010
- B registered mark in 2015
- A can continue use (prior user rights)

**(e) Exhaustion of Rights (Parallel Import)** - Genuine goods sold abroad by proprietor → Can be imported (parallel import)

### Passing Off (Common Law)

**Even Without Registration, Can Sue for Passing Off:**

**Elements:**

1. **Goodwill/Reputation** - Plaintiff has goodwill in mark
2. **Misrepresentation** - Defendant's use causes confusion
3. **Damage** - Actual or likely damage to plaintiff's goodwill

**Remedy:** Injunction + Damages

**Example:**
- Local restaurant "Udipi Café" (unregistered, used for 20 years)
- New restaurant opens nearby: "Uduppi Café"
- Customers confused
- Original owner can sue for passing off (even without registration)

### Remedies (Section 135)

**Civil Remedies:**

1. **Injunction** - Restrain infringement
2. **Damages/Account of Profits** - Compensatory damages OR defendant's profits
3. **Delivery Up** - Infringing goods/labels destroyed

**Criminal Remedies (Section 103-105):**

- **Imprisonment:** Up to 3 years
- **Fine:** Rs. 50,000 to Rs. 2,00,000
- For applying false trademark, selling goods bearing false trademark

**Cognizable Offense** - Police can arrest (with warrant)

## Well-Known Trademarks (Section 11(6)-(9))

**Well-Known Mark:**
- Mark with substantial reputation in India
- Protected even without registration
- Protected across all goods/services (not just registered classes)

**Determination:**
- Registrar determines well-known status (on case-by-case basis)
- Factors (Rule 124):
  - Number of actual/potential consumers
  - Geographical extent of use
  - Duration of use
  - Advertising expenditure
  - Market share
  - Registration in other countries

**Examples:**
- COCA-COLA, McDONALD'S, GOOGLE, APPLE, MICROSOFT

**Protection:**

1. **Against Registration** - Can oppose registration of similar mark in any class
2. **Against Use** - Can restrain use even for unrelated goods/services (dilution)

**Example:**
- ROLEX well-known for watches
- Someone applies ROLEX for bicycles
- Even though different goods → Registrar can refuse (Section 11(6))

## Trademark Maintenance and Renewal

### Section 25: Renewal

**Initial Term:** 10 years from registration

**Renewal:** Indefinitely, every 10 years

**Renewal Application:**
- File within 6 months before expiry
- Can file up to 6 months after expiry (with surcharge)
- If not renewed → Removed from register

**Restoration (Section 60):**
- If mark removed for non-renewal
- Can apply for restoration within 1 YEAR of removal
- Show sufficient cause + pay fee

### Section 47: Removal for Non-Use (5 Years)

**Any Person Can Apply to Remove Mark if:**
- Mark not used for continuous period of 5 years + 3 months
- No bona fide use on goods/services for which registered

**Proprietor's Defense:**
- Prove use in 5 years preceding application
- OR special circumstances preventing use

**Use:**
- Must be bona fide commercial use (not token use)
- On goods/services in India (export use counts)
- By proprietor OR registered user

**Effect of Removal:**
- Mark removed from register
- Becomes available for others

## Trademark Portfolio Strategy

### 1. Selection and Clearance

**Trademark Strength Spectrum:**

**Strongest:**
- **Fanciful** - Invented words (KODAK, XEROX)
- **Arbitrary** - Common words unrelated to goods (APPLE for computers)

**Moderate:**
- **Suggestive** - Suggests quality/characteristic (GREYHOUND for fast buses)

**Weak:**
- **Descriptive** - Describes goods (CREAMY for ice cream) - Not registrable (unless acquired distinctiveness)
- **Generic** - Common name for goods (COMPUTER for computers) - NEVER registrable

**Strategy:** Choose fanciful/arbitrary marks for maximum protection

### 2. Filing Strategy

**Classes:**
- File in all relevant classes (present + future use)
- Consider defensive classes (prevent cybersquatting)

**Priority:**
- File early (first-to-file system in India)
- Consider international filing (Madrid Protocol)

### 3. Policing and Enforcement

**Monitor:**
- TM Journal (weekly) for conflicting applications
- Market for infringing use

**Oppose:**
- File opposition against conflicting applications

**Enforce:**
- Send cease and desist letters
- File infringement suits

### 4. International Protection

**Madrid Protocol:**
- Single application to WIPO (World Intellectual Property Organization)
- Designate multiple countries (India joined Madrid Protocol in 2013)
- Cost-effective for multi-country filing

**Paris Convention:**
- 6-month priority claim
- File in India → 6 months to file abroad claiming India priority date

### Common Mistakes

**1. Using Descriptive Terms** → Weak protection OR unregistrable

**2. Not Conducting Search** → Infringement risk, wasted application fee

**3. Not Filing in All Relevant Classes** → Gaps in protection

**4. Not Using Mark After Registration** → Vulnerable to removal for non-use (Section 47)

**5. Delayed Opposition** → Miss 4-month window

**6. Using ® Before Registration** → Offense under Section 139

**7. Not Renewing** → Mark removed (restoration costly)

**8. Generic Use (Trademark Becoming Generic)** → XEROX preventing "xeroxing"; GOOGLE fighting "googling" as verb

**9. Not Monitoring Journal** → Miss conflicting applications

**10. Inconsistent Use** → Using mark differently than registered (weakens enforcement)
