# Protocol: Execution of Decrees (India)

**Protocol ID**: IL-CPC-EXECUTION  
**Domain**: Civil Procedure  
**Subdomain**: Post-Judgment  
**Version**: 1.0.0  
**Last Updated**: 2025-12-06

## Purpose

This protocol provides comprehensive guidance on execution of decrees and orders under the Code of Civil Procedure 1908, covering procedures, modes of execution, objections, and remedies.

## Activation Triggers

- User asks "How to execute a decree?"
- Decree holder wants to recover decree amount
- Need to attach property for decree satisfaction
- Judgment debtor not paying decree voluntarily
- Execution objections raised
- Cross-objections in execution

## Core Principles

### 1. Execution = Enforcement of Decree

```
Execution (Order XXI):
Process by which court enforces its decree
→ Judgment debtor compelled to satisfy decree
→ Coercive measures if voluntary compliance fails

Decree-Holder = Plaintiff (who obtained favorable decree)
Judgment-Debtor = Defendant (against whom decree passed)

Objective: Translate paper decree into actual relief (money/property/specific act)
```

### 2. Only Decrees Executable (Section 2(2) + Order XXI Rule 11)

```
Executable:
✓ Decrees (money, property, specific performance, injunction)
✓ FINAL decrees (not preliminary - unless expressly made executable)
✓ Orders expressly declared executable by CPC (e.g., Order XXXIX Rule 2A - contempt for injunction breach)

NOT Executable:
✗ Preliminary decrees (until final decree passed)
✗ Orders (generally - except those expressly executable)
✗ Declaratory decrees (no specific relief - only declaration of rights)
```

## WHEN EXECUTION LIES (Order XXI Rule 11)

### Conditions for Execution

```
Order XXI Rule 11: Execution on Application of Decree-Holder

Conditions:

1. DECREE must be FINAL (not preliminary)

2. Decree must have become FINAL:
   - No appeal filed within limitation (90 days expired), OR
   - Appeal filed and dismissed, OR
   - Appeal filed and withdrawn, OR
   - Appellate court modified decree (execute modified decree)

3. Decree must be for SPECIFIC RELIEF:
   - Money
   - Property (movable/immovable)
   - Specific act (specific performance, injunction)

4. Execution application filed within LIMITATION:
   - 12 years from date decree becomes final (Article 136 Limitation Act 1963)

5. Execution in PROPER COURT:
   - Court that passed decree (Rule 3), OR
   - Court to which decree transferred for execution (Rule 4-6)
```

### Courts Which May Execute Decree (Order XXI Rule 3-6)

```
Rule 3: Execution by Court Which Passed Decree (Primary)
Court that passed decree generally executes

Rule 4-6: Transfer for Execution (Secondary)

Transfer If:
(a) Judgment debtor resides outside jurisdiction of decree-passing court
(b) Judgment debtor's property situated outside jurisdiction
(c) Decree-holder requests transfer (for convenience)

Procedure:
1. Decree-passing court transfers decree to court where:
   - Judgment debtor resides, OR
   - Judgment debtor's property situated
2. Transferee court executes as if passed by itself
3. Return warrant after execution complete
```

## PROCEDURE FOR EXECUTION (Order XXI)

### Step 1: File Execution Application (Rule 11)

```
Rule 11: Execution Petition Format

Contents:
1. Decree-holder's name and description
2. Judgment debtor's name and description
3. Decree details (date, court, suit number, amount/relief)
4. Property/assets of judgment debtor to be attached (if known)
5. Mode of execution sought:
   - Attachment and sale of property
   - Arrest and detention
   - Delivery of property
   - Specific performance
6. Verification

Court Fees:
Ad valorem (on decree amount) or fixed (as per state rules)

Timeline:
Within 12 years from decree becoming final (Article 136 Limitation Act)
```

### Step 2: Notice to Judgment Debtor (Rule 22)

```
Rule 22: Notice of Execution

Court issues notice to judgment debtor:
- Show cause why execution should not proceed
- Opportunity to pay decree amount voluntarily
- Appear on specified date

Service:
- Personal service (primary)
- Substituted service (if personal impracticable)
- Proclamation + affixture (if whereabouts unknown)

Time:
- Usually 15-30 days from notice to appear
```

### Step 3: Judgment Debtor's Response

```
Options for Judgment Debtor:

1. PAY DECREE AMOUNT VOLUNTARILY:
   - Full satisfaction → Decree satisfied, execution closed
   - Partial payment → Execution for balance

2. FILE OBJECTION (Order XXI Rule 58, 101):
   - Challenge execution
   - Grounds: Decree satisfied, decree not executable, illegality

3. APPEAR AND REQUEST TIME:
   - Court may give installments (Order XXI Rule 1)
   - Conditional: Judgment debtor shows inability to pay in one go

4. NOT APPEAR / NOT PAY:
   - Court proceeds with execution (coercive measures)
```

### Step 4: Execution Proceedings (Modes of Execution)

```
Modes (depend on nature of decree):

A. Money Decree: Attachment and sale of property
B. Property Decree: Delivery of possession
C. Specific Performance: Compulsion to perform act
D. Injunction: Contempt for breach

Court selects mode based on:
- Nature of decree
- Availability of judgment debtor's assets
- Efficiency of recovery
```

## MODES OF EXECUTION - MONEY DECREES

### A. Attachment and Sale of Property (Rule 41-94)

```
Process:

1. ATTACHMENT (Rule 41-53):
   Court prohibits judgment debtor from transferring/dealing with property

   Procedure:
   - Court issues attachment order
   - Bailiff/process server serves attachment order on judgment debtor
   - Property attached (proclamation posted, if immovable)
   - Attachment prevents sale/transfer by judgment debtor

   Types of Property Attachable:
   ✓ Immovable property (land, buildings)
   ✓ Movable property (goods, vehicles, machinery)
   ✓ Debts owed to judgment debtor (attachment of debts - Rule 46)
   ✓ Shares, securities (Rule 48)
   ✓ Salary/pension (Rule 48 - subject to limits)

   Property EXEMPT from Attachment (Rule 60):
   ✗ Necessary wearing apparel, cooking vessels
   ✗ Tools of trade (up to Rs. 10,000 value)
   ✗ Books, instruments for profession (up to Rs. 10,000)
   ✗ Dwelling house + reasonable land (for agricultural family)
   ✗ Stipends, allowances
   ✗ Certain government pensions
   ✗ 2/3rd of salary (only 1/3rd attachable - Rule 48)

2. PROCLAMATION OF SALE (Rule 54-56):
   - Public notice of sale
   - 30 days minimum from proclamation to sale
   - Details: Property description, upset price, date/time/place of sale

3. SALE (Rule 66-94):
   - Public auction
   - Highest bidder
   - Sale confirmed by court
   - Sale certificate issued to purchaser
   - Judgment debtor dispossessed (if immovable property)

4. DISTRIBUTION OF SALE PROCEEDS (Rule 73):
   - Decree amount paid to decree-holder
   - Costs of execution paid
   - Surplus (if any) returned to judgment debtor

Time Limit for Sale:
- Movable property: Within 15 days of attachment (unless court extends)
- Immovable property: After 30 days from proclamation
```

### B. Arrest and Detention of Judgment Debtor (Rule 30-40)

```
Rule 30: Arrest and Detention for Non-Payment

Conditions:
1. Money decree
2. Judgment debtor has means to pay but refuses
3. Decree-holder applies for arrest
4. Court satisfied arrest necessary

NOT Allowed:
- Arrest of women (unless special circumstances)
- Arrest for civil debt if judgment debtor can show inability to pay

Procedure:
1. Warrant of arrest issued
2. Judgment debtor arrested and brought to court
3. Court examines: Can judgment debtor pay?
   - If YES but refuses → Detention ordered (up to 3 months)
   - If NO (genuinely unable) → Released, explore other modes

Detention Period:
- Maximum 3 months per warrant (Rule 31)
- Court may issue fresh warrant if decree still unsatisfied (but total detention limited)

Release:
- Payment of decree amount → Immediate release
- Inability to pay shown → Released
- Detention period expires → Released

Modern Practice:
- Arrest/detention RARELY used (last resort)
- Courts prefer attachment and sale of property
```

### C. Attachment of Salary/Pension (Rule 48)

```
Rule 48: Attachment of Salary

Limit:
- Maximum 1/3rd of salary attachable (2/3rd exempt)
- Ensures judgment debtor has minimum income for subsistence

Procedure:
1. Court issues order to judgment debtor's employer
2. Employer deducts 1/3rd of salary each month
3. Employer pays deducted amount to court
4. Court pays decree-holder
5. Continues till decree fully satisfied

Effect:
- Gradual satisfaction (installments)
- Judgment debtor continues working
- No immediate hardship

Similar: Pension attachment (with limits to protect minimum subsistence)
```

### D. Attachment of Debts (Rule 46)

```
Rule 46: Attachment of Debts Owed to Judgment Debtor

If third party owes money to judgment debtor:
→ Court attaches debt (prohibits third party from paying judgment debtor)

Procedure:
1. Decree-holder identifies third party debtor
2. Court issues order to third party (garnishee order)
3. Third party prohibited from paying judgment debtor
4. Third party pays court instead
5. Court pays decree-holder

Example:
Judgment debtor is landlord; tenant owes rent
→ Court attaches rent
→ Tenant pays rent to court (not to judgment debtor)
→ Court pays decree-holder

Third Party's Rights:
- Can dispute debt (claim debt not owed)
- Can show debt already paid
- Court decides (mini-trial on debt existence)
```

## MODES OF EXECUTION - PROPERTY DECREES

### Delivery of Immovable Property (Rule 95-96)

```
Decree for Possession of Immovable Property:

Procedure:
1. Decree-holder files execution application
2. Court issues order for delivery of possession
3. Bailiff/court officer delivers possession to decree-holder
4. If judgment debtor refuses to vacate:
   - Court officer removes judgment debtor's belongings
   - Hands over possession to decree-holder
5. Attachment if judgment debtor attempts re-entry (contempt)

Forcible Entry:
- Court officer may break open locks if necessary (with court permission)
- Police assistance sought if resistance expected

Mesne Profits (Rule 96):
- If decree includes mesne profits (rent for wrongful possession)
- Calculated by Commissioner OR court
- Recovered like money decree
```

### Delivery of Movable Property (Rule 97)

```
Decree for Specific Movable Property:

Procedure:
1. Court issues order for delivery
2. Bailiff seizes property from judgment debtor
3. Delivers to decree-holder
4. If judgment debtor hides/disposes property:
   - Court may order attachment of other property (value equivalent)
   - OR order payment of value + costs

Example:
Decree: "Defendant to return plaintiff's car (specific vehicle)"
Execution: Bailiff seizes car from defendant, delivers to plaintiff
If car sold/destroyed: Defendant pays market value of car
```

## MODES OF EXECUTION - SPECIFIC PERFORMANCE & INJUNCTION

### Specific Performance Decree (Rule 98-101)

```
Decree Directing Judgment Debtor to Perform Specific Act:

Example:
"Defendant to execute sale deed in favor of plaintiff within 30 days"

Procedure:
1. Decree-holder applies for execution
2. Court issues notice to judgment debtor
3. Options:
   (a) Judgment debtor performs act (executes sale deed)
       → Decree satisfied
   (b) Judgment debtor refuses to perform
       → Court appoints Commissioner/Registrar to execute on behalf of judgment debtor
       → Deemed as if judgment debtor executed (Section 34 Specific Relief Act)

Effect of Commissioner's Execution:
- Binding on judgment debtor
- Conveyance effective as if judgment debtor signed

Alternative:
- If performance impossible OR judgment debtor deliberately defeated
  → Decree-holder may apply for compensation in lieu
  → Court orders judgment debtor to pay value (like money decree)
```

### Injunction Decree (Order XXXIX Rule 2A, Section 94)

```
Decree Restraining Judgment Debtor from Doing Act:

Example:
"Defendant perpetually restrained from constructing on plaintiff's land"

Execution/Enforcement:
1. Injunction decree issued
2. If judgment debtor violates (does prohibited act):
   → Decree-holder files contempt application
   → Section 94 CPC / Order XXXIX Rule 2A: Contempt of court
   → Court may:
     (a) Fine judgment debtor
     (b) Imprison judgment debtor (up to 6 months)
     (c) Attach and sell judgment debtor's property

Contempt Procedure:
- Show cause notice to judgment debtor
- Judgment debtor explains violation (or denies)
- If violation proved → Contempt punishment

Prevention:
- If imminent violation → Decree-holder seeks attachment of property as preventive measure
```

## OBJECTIONS TO EXECUTION (Order XXI Rule 58, 101)

### Types of Objections

```
Rule 58: Objections to Execution BY JUDGMENT DEBTOR

Grounds:
1. Decree satisfied (payment already made)
2. Decree executed (in full or part) already
3. Decree stayed by appellate court (appeal pending + stay granted)
4. Decree set aside/modified
5. Decree not executable (preliminary decree, declaratory decree)
6. Decree-holder not entitled to execute (not party to decree)
7. Property sought to be attached exempt (Rule 60)
8. Execution barred by limitation (12 years expired)

Procedure:
- Judgment debtor files objection application
- Affidavit stating grounds
- Court hears parties
- Decides objection

Effect:
- If objection sustained → Execution dismissed/stayed
- If objection rejected → Execution proceeds
```

### Rule 101: Objections to Execution BY THIRD PARTY

```
Third Party = Person not party to decree, claiming interest in property attached

Example:
Decree-holder attaches property; third party claims "This is MY property, not 
judgment debtor's"

Grounds:
1. Property belongs to third party (not judgment debtor)
2. Third party has lien/charge on property
3. Third party is tenant/occupant with rights
4. Property jointly owned (third party co-owner)

Procedure:
1. Third party files objection (within prescribed time - usually 30 days from attachment)
2. Affidavit + documents proving ownership/interest
3. Court hears third party and decree-holder
4. Decides:
   (a) Third party objection valid → Attachment released (as to third party's share)
   (b) Third party objection invalid → Attachment continues, sale proceeds

Separate Suit:
If third party's claim involves detailed investigation:
→ Court may direct third party to file separate suit
→ Execution stayed/continued pending suit
```

### Cross-Objections

```
If judgment debtor or third party files objection AND decree-holder disputes:
→ Decree-holder files CROSS-OBJECTION
→ Supports execution, disputes objection grounds

Example:
Judgment Debtor's Objection: "Decree satisfied by payment on DD/MM/YYYY"
Decree-Holder's Cross-Objection: "No payment received; payment claim false"

Court decides objection + cross-objection together
```

## SATISFACTION OF DECREE (Rule 2)

### Decree Satisfaction

```
Rule 2: Satisfaction Entry

Decree Satisfied When:
1. Decree amount paid in full (voluntary OR through execution)
2. Property delivered (in property decree)
3. Specific act performed (in specific performance decree)
4. Parties compromise post-decree (satisfaction agreement)

Procedure:
1. Judgment debtor pays decree amount to court OR decree-holder
2. Decree-holder acknowledges satisfaction (filing application in court)
3. Court makes SATISFACTION ENTRY in decree register
4. Decree marked as "Satisfied"

Effect:
- Decree cannot be executed again
- Judgment debtor discharged from liability
- Execution proceedings closed

Partial Satisfaction:
- Payment of part of decree amount
- Court notes partial satisfaction
- Execution for balance continues

Certificate of Satisfaction:
- On request, court issues certificate stating decree satisfied
- Useful for judgment debtor (proof of satisfaction)
```

## PIECEMEAL EXECUTION

```
Piecemeal Execution = Executing decree in parts (multiple execution applications)

Permitted:
✓ Decree for multiple reliefs (can execute each separately)
✓ Decree for money + interest (can execute principal, then interest)
✓ Decree for multiple properties (can attach one, then another)

NOT Permitted:
✗ Unnecessarily splitting decree to harass judgment debtor
✗ Multiple executions for same relief (all at once preferred)

Example:
Decree: "Defendant to pay Rs. 10 lakhs + interest @ 9% p.a."
Execution 1: Attach property worth Rs. 5 lakhs, sell, recover Rs. 5 lakhs
Execution 2 (if needed): Attach another property, recover balance Rs. 5 lakhs + interest

Court discourages piecemeal if decree-holder can execute in one go
```

## LIMITATION FOR EXECUTION (Article 136 Limitation Act 1963)

```
Article 136: 12 YEARS from date decree becomes final

"Final" means:
- No appeal filed within limitation (90 days expired), OR
- Appeal decided

Calculation:
Decree date: 01/01/2020
No appeal filed (90 days expired): 01/04/2020
Execution limitation: 01/04/2020 + 12 years = 01/04/2032

If Appeal:
Decree date: 01/01/2020
Appeal filed: 15/02/2020
Appeal dismissed: 01/05/2021
Execution limitation: 01/05/2021 + 12 years = 01/05/2033

Extension:
- Limitation for execution can be extended (Section 5 Limitation Act) if:
  * Execution stayed by court order (period of stay excluded)
  * Bona fide mistake in execution
- BUT generally strict (12 years is long period; courts expect diligence)

Fresh Period (Article 136 Explanation):
Each execution application starts FRESH 12-year period
- First execution: 01/04/2020 to 01/04/2032
- If first execution dismissed (procedural defect): Fresh execution application within 12 years from dismissal
```

## COMMON SCENARIOS

### Scenario 1: Attachment and Sale of Immovable Property

```
Facts:
- Money decree for Rs. 50 lakhs
- Judgment debtor owns house (market value Rs. 70 lakhs)

Execution:
1. Decree-holder files execution application (attachment of house)
2. Court issues attachment order
3. Proclamation of sale (30 days' notice)
4. Sale date fixed
5. Auction held; highest bidder Rs. 68 lakhs
6. Sale confirmed by court
7. Sale certificate issued to purchaser
8. Sale proceeds distributed:
   - Decree amount: Rs. 50 lakhs to decree-holder
   - Execution costs: Rs. 3 lakhs
   - Surplus: Rs. 15 lakhs to judgment debtor
9. Purchaser gets possession

Effect:
- Decree satisfied
- Judgment debtor loses house but gets surplus
```

### Scenario 2: Third Party Objection

```
Facts:
- Decree-holder attaches property (land)
- Third party claims land belongs to him (not judgment debtor)

Objection:
1. Third party files objection under Order XXI Rule 101
2. Produces sale deed showing he purchased land from judgment debtor 2 years ago
3. Decree-holder disputes: "Sale was fraudulent to defeat decree"

Court Hearing:
- Third party proves bona fide purchase (paid value, registered sale deed)
- Decree-holder fails to prove fraud

Court Order:
- Third party objection sustained
- Attachment of land released
- Decree-holder must find other property of judgment debtor

Alternative (if fraud proved):
- Sale deemed fraudulent (to defeat creditors)
- Attachment continues
- Land sold, third party's "purchase" set aside
```

### Scenario 3: Decree Satisfied by Payment

```
Facts:
- Decree for Rs. 10 lakhs
- Judgment debtor pays Rs. 10 lakhs to decree-holder outside court

Satisfaction:
1. Decree-holder files application stating decree satisfied
2. Affidavit: Received Rs. 10 lakhs on DD/MM/YYYY
3. Court makes satisfaction entry
4. Decree marked "Satisfied"

Later:
- If judgment debtor wants proof: Apply for certificate of satisfaction
- Court issues certificate (useful if decree-holder later denies payment)

If Decree-Holder Denies Payment:
- Judgment debtor shows evidence (receipt, bank transfer, decree-holder's acknowledgment)
- Court decides (may direct decree-holder to file affidavit)
```

### Scenario 4: Execution of Specific Performance Decree

```
Facts:
- Decree: "Defendant to execute sale deed for property in favor of plaintiff within 30 days"
- Defendant refuses to execute

Execution:
1. Plaintiff files execution application
2. Court notices defendant
3. Defendant refuses
4. Court appoints Commissioner (or Registrar) to execute sale deed on behalf of defendant
5. Commissioner executes sale deed
6. Sale deed registered
7. Plaintiff becomes owner

Effect (Section 34 Specific Relief Act):
- Commissioner's execution deemed as defendant's execution
- Binding on defendant
- Property transferred to plaintiff

Alternative (if defendant destroyed property/made performance impossible):
- Plaintiff applies for compensation in lieu
- Court orders defendant to pay market value of property + costs
```

## CASE LAW

### 1. Sardar Govindrao Mahadik v. Devi Sahai (SC 1982)
```
Facts: Property exempt from attachment under Order XXI Rule 60
Held:
- Dwelling house of agricultural family exempt (reasonable area)
- Purpose: Protect judgment debtor's basic subsistence
- Decree-holder must attach other property (if available)
```

### 2. Manoharlal v. Shantabai (SC 1969)
```
Facts: Execution barred by limitation (12 years expired)
Held:
- Article 136 Limitation Act: 12 years from decree becoming final
- Decree becomes final when appeal time expires OR appeal decided
- After 12 years → Execution barred (decree unenforceable)
```

### 3. Rameshwar Dayal v. Suit Property (SC 1977)
```
Facts: Third party objection to execution; property claimed by third party
Held:
- Third party (not party to decree) can object under Order XXI Rule 101
- Burden on third party to prove ownership/interest in property
- If third party proves → Attachment released (as to third party's share)
- If third party fails → Attachment continues
```

### 4. Popat v. Shiv Kumar (SC 2007)
```
Facts: Piecemeal execution; decree-holder filing multiple execution applications
Held:
- Piecemeal execution permitted (if bona fide - decree cannot be executed in one go)
- BUT court discourages if appears to harass judgment debtor
- Each execution application starts fresh limitation period (Article 136)
```

## SUMMARY CHECKLIST

### Execution Application Checklist:

```
☐ Decree final (no appeal OR appeal decided)
☐ Within 12 years from decree becoming final
☐ File execution application (Order XXI Rule 11)
☐ State mode of execution (attachment/arrest/delivery)
☐ Pay court fees
☐ Identify judgment debtor's property/assets (if known)
☐ Serve notice on judgment debtor
☐ Attend execution proceedings
☐ If objection raised → File cross-objection
☐ After satisfaction → File for satisfaction entry
```

### Judgment Debtor Defense Checklist:

```
☐ Check decree (final? correctly drawn?)
☐ Check limitation (12 years expired?)
☐ If decree satisfied → File objection (Order XXI Rule 58)
☐ If property exempt → File objection (Order XXI Rule 60)
☐ If decree stayed → Produce stay order
☐ If unable to pay → Request installments
☐ If third party claims property → Third party files objection (Rule 101)
```

## COMMON MISTAKES

```
Mistake 1: Attempting to Execute Preliminary Decree
❌ File execution for preliminary decree (partition)
✓ Wait for final decree (after actual partition); only final decree executable

Mistake 2: Not Identifying Judgment Debtor's Assets
❌ File execution application without knowing judgment debtor's property
✓ Investigate before filing (judgment debtor's land, bank accounts, salary)

Mistake 3: Missing Limitation (12 Years)
❌ File execution after 12 years from decree becoming final
✓ File within 12 years (strict; difficult to extend after 12 years)

Mistake 4: Attaching Exempt Property
❌ Attempt to attach judgment debtor's dwelling house (agricultural family)
✓ Check Order XXI Rule 60 exemptions; attach non-exempt property

Mistake 5: Not Filing Satisfaction Entry
❌ Decree satisfied (payment received) but no court entry
✓ File satisfaction application in court (prevents future execution claims)

Mistake 6: Ignoring Third Party Objection
❌ Proceed with sale despite third party claiming ownership
✓ Adjudicate third party objection first (Order XXI Rule 101)

Mistake 7: Executing Without Court Permission
❌ Decree-holder directly seizes judgment debtor's property
✓ All execution through COURT (not private enforcement)
```
