# Chambers of SHB for Claude and Claude Code

Load this repository as a prompt/plugin directory. Start with `skills/chambers-shb-orchestrator/SKILL.md` for routing and safety rules.

For court-data work, also load `skills/bharat-courts/SKILL.md` and install `bharat-courts[ocr,archive]`.

For Claude Desktop MCP use, add the local server config from `.mcp.json` or install the MCPB manifest in `manifest.json`.

Core instruction: use the narrowest applicable agent, command, skill, or protocol. Never invent citations, procedural rules, or legal facts.
