# Instructions

Given input facts:

1. Identify nature of issue:
   - Fraud → IPC 420
   - Breach of trust → IPC 406
   - Assault → IPC 351/352
   - Property dispute → CPC

2. Select ONLY relevant sections

3. Output format:

LEGAL_SECTIONS:
- Section X IPC – Reason
- Section Y CrPC – Reason
