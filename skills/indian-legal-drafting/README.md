# Indian Legal Drafting Skill

## How to Use

Provide:

- document_type: criminal_complaint
- facts: Detailed case facts
- parties: Names and roles
- relief: Desired outcome

## Example Input

{
  "document_type": "criminal_complaint",
  "facts": "The accused took money and refused repayment...",
  "parties": "Complainant: A, Accused: B",
  "relief": "Initiate legal action"
}

## Output

- Court-ready legal draft
- Relevant IPC/CrPC/CPC sections
- Structured format
