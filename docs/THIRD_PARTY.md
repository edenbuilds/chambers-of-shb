# Third-Party Materials

Chambers of SHB combines and normalizes materials supplied to this repository by the user.

Included source licenses are preserved in `docs/source-licenses/`:

- `indian-law-plugin-LICENSE`
- `company-drafting-LICENSE`
- `indian-legal-drafting-LICENSE.txt`

The supplied `NCLT-Skills-main.zip` archive contained proprietary Anthropic license terms restricting extraction and redistribution. Those files are not redistributed in Chambers of SHB. Chambers of SHB includes an original `nclt-pdf-workflow` skill instead, covering local PDF and tribunal filing workflow practices without copying restricted source text.

Legal-Lens Pro influenced the contract-risk and legal-aid orientation concepts in this repo. No source code was copied from that repository because no redistribution license was present in the cloned source.
