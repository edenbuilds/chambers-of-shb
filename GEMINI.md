# Chambers of SHB for Gemini

Use this repository as the legal prompt pack and local reference base. Read `skills/chambers-shb-orchestrator/SKILL.md` first for routing, safety, drafting, and verification rules.

When MCP is available, run the local server from `.mcp.json` to search protocols, fetch commands, retrieve agents, and produce local drafting artifacts.

For Indian court data, install `bharat-courts[ocr,archive]` and use `skills/bharat-courts/SKILL.md`.
